# cns-init-lab

本仓库用于归档和存放多个独立 HTML 页面。这些页面以静态文件形式组织，不绑定特定域名或网站，可作为通用 HTML 资源库使用。

## 目录说明

```
cns-init-lab/
├── pages/          # 存放所有 HTML 页面文件
├── assets/         # 页面引用的资源文件（图片、样式、脚本等）
└── README.md       # 本文件
```

- `pages/`：每个子目录或文件对应一个独立 HTML 页面，按功能或主题归类。
- `assets/`：页面所需的公共资源，包括 CSS、JavaScript、图像等。

## 页面归档说明

- 所有页面均为纯静态 HTML，无需后端服务即可在浏览器中直接打开。
- 页面之间相互独立，无依赖关系，可单独使用或引用。
- 归档内容不涉及任何特定域名、品牌或商业推广信息。

## 使用方式

1. 克隆或下载本仓库到本地。
2. 使用浏览器直接打开 `pages/` 目录下的 HTML 文件。
3. 也可将整个目录部署到任意静态托管服务中。

## 维护说明

- 欢迎提交 Pull Request 添加新的 HTML 页面或改进现有页面。
- 请确保新页面不包含恶意代码、外部跟踪脚本或商业广告。
- 资源文件请统一放入 `assets/` 目录，避免引用外部不可控资源。
- 如有问题或建议，请通过 Issue 反馈。
